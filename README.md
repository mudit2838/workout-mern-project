## API endpoints
GET - /workout  --> all the workout docs 
POST - /workouts --> create a new workout doc
GET - /workouts/:id --> get a single workout doc
DELETE - /workouts/:id --> delete a single workout doc
PATCH - /workouts/:id --> udate a single workout
